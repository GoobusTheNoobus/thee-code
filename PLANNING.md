# Planning
This markdown sheet will be used for planning the language.

Our language's philosophy:
- Less typing
- Simple syntax
- Braces are necessary, indents are optional
- There should be one way to do something

# Syntax Style
```
let num1 = 67
let num2 = num1 * 2 + 5 / 4

prt "Number 1: " num1 " Number 2: " num2 nl
prt "Diff: " math.abs(num2 - num1) nl

```
